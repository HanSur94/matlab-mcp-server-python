    **Step 2: Create `src/matlab_mcp/auth/middleware.py`:**
    Pure ASGI middleware class (NOT BaseHTTPMiddleware â€” per user decision, to avoid Starlette streaming double-send bug).

    Key implementation details:
    - `_BYPASS_PATHS = {"/health"}` â€” set of paths that skip auth
    - Class `BearerAuthMiddleware` with `__init__(self, app: ASGIApp) -> None`:
      - Store `self._app = app`
      - Read token: `self._token: str | None = os.environ.get("MATLAB_MCP_AUTH_TOKEN")`
    - `async def __call__(self, scope: Scope, receive: Receive, send: Send) -> None`:
      1. If `scope["type"] != "http"` -> pass through (`await self._app(scope, receive, send)`)
      2. If `scope.get("path", "")` in `_BYPASS_PATHS` -> pass through
      3. If `self._token is None` -> pass through (auth disabled, no token configured)
      4. If `scope.get("method") == "OPTIONS"` -> pass through (CORS pre-flight)
      5. Extract Authorization header from `scope["headers"]` (list of 2-tuples of bytes)
      6. Parse "Bearer " prefix (case-insensitive startswith on the value)
      7. Compare with `hmac.compare_digest(provided_token, self._token)` â€” both must be `str`
      8. If match -> pass through
      9. If no match -> send 401 response:
         - Body: `json.dumps({"error": "unauthorized", "message": "Valid Bearer token required"}).encode()`
         - Headers: `content-type: application/json`, `content-length: <len>`, `www-authenticate: Bearer`
         - Use raw ASGI: `await send({"type": "http.response.start", "status": 401, "headers": [...]})` then `await send({"type": "http.response.body", "body": body})`

    Import: `from __future__ import annotations`, `import hmac`, `import json`, `import os`, `import logging`
    Add `logger = logging.getLogger(__name__)` per project convention.
