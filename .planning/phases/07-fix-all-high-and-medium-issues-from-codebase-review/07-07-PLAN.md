**Issue 38 — Replace asyncio.sleep(0.5) races (tests/test_executor_extra.py):**

Find all `await asyncio.sleep(0.5)` calls in `test_executor_extra.py` (should be around lines 476, 492, 505, 516, 529, 554).

Replace each with event-based synchronization. The pattern depends on what the sleep is waiting for:

For background task completion waits:
```python
# Instead of: await asyncio.sleep(0.5)
# Use: wait for executor's background tasks to complete
if executor._background_tasks:
    await asyncio.wait_for(
        asyncio.gather(*list(executor._background_tasks), return_exceptions=True),
        timeout=5.0,
    )
```

For tests that just need to yield control:
```python
# Instead of: await asyncio.sleep(0.5)
# Use: await asyncio.sleep(0) to yield, then check condition
await asyncio.sleep(0)
```

For tests waiting for a condition:
```python
# Poll with short interval and reasonable timeout
for _ in range(50):
    if condition_met():
        break
    await asyncio.sleep(0.1)
else:
    pytest.fail("Condition not met within 5s")
```
